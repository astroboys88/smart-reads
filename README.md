# Smart Reads — AI Article Summarizer

> Baca cerdas, hemat waktu. Paste URL atau teks → AI merangkum jadi 3 poin utama + key takeaways + actionable insight.

**100% web-based. Single HTML file. No server needed.**

## What it does

- **URL Fetch** — Masukkan link artikel, AI auto-extract content dan summarize
- **Text Paste** — Paste teks panjang langsung, AI proses dan ringkas
- **3-Point Summary** — 3 kalimat ringkas yang captures inti artikel
- **Key Takeaways** — 3-5 insight penting worth untuk diingat
- **Actionable Insight** — 1 langkah konkret yang bisa langsung di-execute hari ini
- **Reading History** — Simpan & replay hasil summarize sebelumnya
- **Xiaomi MiMo API** — Powered by MiMo-8B model

## Tech Stack

- Pure HTML + CSS + Vanilla JS (no build)
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Phosphor Icons](https://phosphoricons.com/) via CDN
- [Xiaomi MiMo API](https://platform.xiaomimimo.com) — AI inference

## Quick Start

1. **Get MiMo API Key** → Apply di [100T Program](https://100t.xiaomimimo.com) (free credits)
